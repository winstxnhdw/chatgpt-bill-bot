Telegram cron bot for announcing the bill to members in a chat group.

## Commands

### Setup

Install all dependencies.

```bash
yarn
```

Run your application.

```bash
yarn dev
```

### Build

Build a human-readable bundle of your Node application. For debugging purposes.

```bash
yarn build test
```
